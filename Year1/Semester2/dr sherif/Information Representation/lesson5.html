What is Resolution?

Resolution measures the number of pixels (picture elements) in an image or on a display.
Higher resolution = more pixels = sharper and clearer image
Used for: monitors, cameras, TVs, printers, smartphones...

Monitor Size & Pixel Density (PPI)

Same resolution on larger screen → pixels are spread out → image looks less sharp
PPI (pixels per inch) determines perceived sharpness:
• Smaller screen + same resolution = higher PPI = sharper image
• Larger screens need higher resolution to maintain quality

Types of Resolution

1. Image resolution
Pixel density in the image file (PPI)
2. Screen resolution
Total pixels the display can show (e.g. 1920×1080 = Full HD)
Common modern standards:
• 1920×1080 → Full HD
• 2560×1440 → 2K/QHD
• 3840×2160 → 4K UHD

Data Packets in Networks

Large data is broken into small packets (typically ~64 KiB) for transmission.
Benefits of packetization: • Packets can take different routes
• More efficient use of network
• Better handling of congestion/failures

Packet Structure

Typical packet contains:
  • Header — source/destination IP, sequence number, packet size...
  • Payload — the actual data
  • Trailer — error checking (CRC), end marker
MAC addresses change at each hop
IP addresses remain the same from start to finish

Packet Switching

Modern internet uses packet switching: • Each packet travels independently
• Routers decide best path at each hop
• Packets may arrive out of order → reassembled at destination

Advantages vs Disadvantages

Advantages Disadvantages
No dedicated line neededPackets can arrive out of order
Very resilient (packets reroute around failures)Reassembly delay at destination
Efficient use of bandwidthNot ideal for real-time streaming (possible frame loss)
Supports different device speedsOverhead from headers

Transmission Types & USB

Main categories:
  • Direction: Simplex / Half-duplex / Full-duplex
  • Timing: Synchronous / Asynchronous
  • Method: Serial / Parallel
USB (most common today): • Serial, full-duplex, asynchronous
• Auto-detection & driver loading
• Backward compatible
• Max practical cable length ~5m
