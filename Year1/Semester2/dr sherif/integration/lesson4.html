Three Ways to Insert CSS

Mastering External, Internal, and Inline styles

1. External CSS

Best for styling an entire website — change one file to update everything.

  • Saved with .css extension
  • No HTML tags allowed inside
  • Linked in the <head> using <link>
<link rel="stylesheet" href="mystyle.css">

2. Internal CSS

Used when a single page needs unique styling.

  • Defined inside <style> element
  • Placed in the <head> section
<head>
  <style>
    body { background-color: linen; }
  </style>
</head>

3. Inline CSS

Applies unique style to a single element directly.

  • Added via the style attribute
  • Highest priority (overrides others)
<h1 style="color:blue; text-align:center;">This is Blue</h1>

Cascading Order & Priority

When styles conflict, CSS follows this priority (highest to lowest):

  1. Inline style (highest priority)
  2. Internal or External (whichever is read last)
  3. Browser default (lowest)

Important Syntax Note

Critical: Never put a space between the value and its unit!

❌ Wrong: margin-left: 20 px;
✅ Correct: margin-left: 20px;
