HTML Attributes

Enhancing HTML elements with additional information and properties

What are Attributes?

Attributes provide additional information about HTML elements. They are essential for defining how elements behave or where they link to.

  • All HTML elements can have attributes.
  • Attributes are always specified in the start tag.
  • They usually come in name/value pairs like name="value".

The href Attribute

The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to.

<a href="https://www.w3schools.com">Visit W3Schools</a>

Image Attributes: src, width, height

The <img> tag embeds images. It requires the src attribute to specify the path to the image.

<img src="img_girl.jpg" width="500" height="600">
URL Types for src:
  • Absolute: src="https://site.com/img.jpg"
  • Relative: src="img_girl.jpg"

Tip: Relative URLs are almost always preferred — they won't break when the domain changes.

The alt Attribute

The required alt attribute provides alternate text if the image cannot be displayed. Essential for accessibility and slow connections.

<img src="img.jpg" alt="Girl with a jacket">

The style Attribute

Adds inline CSS styles directly to an element (color, font, size, etc.).

<p style="color:red;">This is a red paragraph.</p>

The lang Attribute

Should always be included in the <html> tag to declare the document language.

<html lang="en-US">

The title Attribute

Provides extra information shown as a tooltip on hover.

<p title="I'm a tooltip">Mouse over me!</p>

🚀 Practice Lab

Complete these quick tasks to reinforce HTML attributes:

Exercise 1: Add a tooltip to the paragraph that says "Hint".

<p ="Hint">Hover over me!</p>

Exercise 2: Set image dimensions to 250 × 400 pixels.

<img src="pic.jpg" ="250" ="400">

Exercise 3: Fix the incorrect attribute name for the link.

<a ="https://google.com">Search</a>
