CSS Positioning

Controlling the layout, stacking order, and alignment of elements.

The Five Position Types

The position property specifies the type of positioning method used for an element.

Type Description
Static Default. Follows the normal document flow.
Relative Positioned relative to its normal position without affecting siblings.
Absolute Positioned relative to the nearest non-static ancestor. Removed from flow.
Fixed Positioned relative to the viewport. Stays in place during scroll.
Sticky Switches between relative and fixed based on the user's scroll position.

Relative Positioning

Shifts an element using top, bottom, left, or right without moving the elements around it.

.box {
  position: relative;
  top: 20px;
  left: 30px;
}

Fixed Positioning

The element is removed from the document flow and stays at a specific spot on the screen even when you scroll.

.header {
  position: fixed;
  top: 0;
  width: 100%;
}

Key Differences to Remember

  • Normal Flow: Only static and relative keep the element's space in the document layout.
  • Ancestors: absolute positioning requires a parent with a position other than static to be contained.
  • Offsets: The top, bottom, left, and right properties will not work unless position is set first.
