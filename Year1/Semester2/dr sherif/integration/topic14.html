PHP: Server-Side Power

Mastering Dynamic Web Logic & Database Integration

1. Variables & Data

In PHP, variables start with the $ sign. Unlike other languages, you don't need a command to declare them; they are created when you assign a value.

$name = "Sherif";
$age = 25;

Core Rules:

  • Must start with a letter or underscore.
  • Cannot start with a number.
  • Case-Sensitive: $age and $AGE are different.

2. GET vs POST

These are superglobals used to collect form data.

// Visible in URL (Limited)
$_GET['user'];

// Invisible/Secure (Unlimited)
$_POST['pass'];

Use POST for sensitive data like passwords!

3. Arrays & Logic

PHP handles collections of data using Indexed Arrays and Associative Arrays (key-value pairs).

// Associative Array
$prices = ["Apple" => "2.00"];

Control the flow using if/else statements and loops to repeat actions.

4. Database Integration (MySQL)

PHP connects your website to a database to store and retrieve records. You can use MySQLi or PDO.

$sql = "INSERT INTO MyGuests (firstname, lastname) VALUES ('John', 'Doe')";
if (mysqli_query($conn, $sql)) {
    echo "New record created successfully";
}
