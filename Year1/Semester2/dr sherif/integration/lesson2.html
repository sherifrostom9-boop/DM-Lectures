CSS Basics

Understanding Cascading Style Sheets and the power of Selectors

What is CSS?

CSS (Cascading Style Sheets) is the language used to style an HTML document. It describes how HTML elements should be displayed on screen, print, or other media.

  • Allows central control of appearance across multiple pages
  • External stylesheets are saved in .css files
  • Replaced the messy inline formatting common in early HTML

CSS Syntax

A CSS rule consists of a selector and a declaration block.

h1 {
  color: white;
  text-align: center;
}

The selector targets the element(s), declarations are separated by semicolons.

Types of Selectors

Element Selector — targets tag names

p { margin: 1em 0; }

ID Selector — unique element (#)

#header { background: navy; }

Class Selector — reusable style (.)

.btn-primary { background: var(--primary); }

Universal Selector — everything (*)

* { box-sizing: border-box; }

Grouping: Apply same rules to multiple selectors

h1, h2, h3 { font-family: "Inter", sans-serif; }
