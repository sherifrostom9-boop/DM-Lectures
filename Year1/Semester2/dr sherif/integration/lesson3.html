CSS Syntax & Selectors

Breaking down how CSS rules are constructed and applied

The CSS Rule-Set

A complete CSS rule consists of a selector followed by a declaration block.

h1 {
  color: blue;
  font-size: 16px;
}
  • Selector — targets which element(s) to style
  • Declaration block — enclosed in curly braces { }
  • Declaration — property: value; pairs separated by semicolons

ID Selector (#)

Selects a unique element using its id attribute.

  • ID must be unique per page
  • Use # followed by the ID name
  • Cannot start with a number
#main-title { font-weight: bold; }

Class Selector (.)

Selects elements that have a specific class attribute.

  • Reusable across multiple elements
  • Use . followed by the class name
  • Cannot start with a number
.highlight { background: yellow; }

Universal Selector (*)

Selects every element on the page.

* { margin: 0; padding: 0; }

Grouping Selector

Apply the same declarations to multiple selectors by separating them with commas.

h1, h2, .intro, #hero {
  text-align: center;
}

🚀 Syntax Lab

Complete the CSS syntax correctly for each rule.

1. Select the element with id="main-title":

{ color: pink; }

2. Select all elements with class "bold-text":

{ font-weight: bold; }

3. Complete the declaration (property and value):

h1 { : ; }

4. Group h1 and h2 selectors:

h1 h2 { text-decoration: underline; }
