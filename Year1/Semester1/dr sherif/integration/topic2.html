Lesson 02

HTML Attributes

Providing additional information and properties to HTML elements.

What are Attributes?

  • All HTML elements can have attributes[cite: 8].
  • Attributes provide additional information about elements[cite: 9].
  • They are always specified in the start tag[cite: 10].
  • They usually come in name/value pairs: name="value"[cite: 11].

The href Attribute

Used with the <a> tag to define hyperlinks. It specifies the URL of the page the link goes to[cite: 14, 15].

<a href="https://www.w3schools.com">Visit W3Schools</a>

The src Attribute

Used with the <img> tag to embed images. It specifies the path to the image file[cite: 35, 36].

<img src="img_girl.jpg">
Tip: Relative URLs (like "img_girl.jpg") are usually better because they won't break if you change your domain[cite: 67, 73].

Size & Alt Attributes

Images should include width and height in pixels[cite: 84]. The alt attribute provides alternate text if the image fails to load[cite: 88].

<img src="img_girl.jpg" alt="Girl with a jacket" width="500" height="600">

The style Attribute

Used to add inline styles to an element, such as color, font, and size[cite: 93].

<p style="color:red;">This is a red paragraph.</p>

The lang & title Attributes

Lang: Declares the language of the web page inside the <html> tag[cite: 100].

<html lang="en-US">

Title: Defines extra info shown as a tooltip on hover[cite: 123, 124].

<p title="I'm a tooltip">Hover over me!</p>

Chapter Summary

  • Attributes provide additional info[cite: 128].
  • href is for links[cite: 129].
  • src, alt, width, and height are for images[cite: 130, 131, 132].
  • style is for inline design[cite: 133].
  • lang assists search engines and browsers[cite: 101, 134].
