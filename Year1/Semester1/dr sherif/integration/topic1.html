What is HTML?

Content structuring, tag semantics, and W3C standards.

The Definition

HTML (HyperText Markup Language) is the standard markup language for creating Web pages. It describes the structure of a Web page and consists of a series of elements that tell the browser how to display content.

The Web Browser

A software that allows a user to display web pages. Its main purpose is to render HTML code and translate it into text, videos, images, and sound.

  • Interprets HTML & Scripts
  • Manages History & Bookmarks
  • Stores Cookies for sessions

Structure, Presentation & Behavior

Structure (HTML) Presentation (CSS) Behavior (JS)
The "Bones": What content it has (Headings, Paragraphs). The "Skin": How it looks (Fonts, Colors, Alignments). The "Brain": How it works (Alerts, Buttons, Interactions).

Standard Page Structure

Every professional website follows this nesting logic:

<!-- Document Type -->
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Visible Heading</h1>
    <p>Visible Paragraph</p>
  </body>
</html>

Essential Tags

  • <!DOCTYPE html>: Informs the browser about the HTML version (HTML5).
  • <html>: The root element of the page.
  • <head>: Contains metadata (not visible to users).
  • <body>: Contains all visible content (images, text, links).
