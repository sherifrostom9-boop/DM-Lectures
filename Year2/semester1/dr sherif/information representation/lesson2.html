Geometric Transformations

CSS Transform Properties and 2D/3D Transformations

📐 What are Geometric Transformations?

Geometric transformations of the plane and space involve changes in the position, size, or shape of geometric figures.

In web development, CSS transforms allow you to modify elements in 2D or 3D space without affecting the normal document flow.

🔑 Key Transformation Concepts

📍 Translation

Moves a figure along a fixed line without changing its size or shape.

🪞 Reflection

Flips a figure across a line, creating a mirror image while preserving size and shape.

🔄 Rotation

Turns a figure around a fixed point, maintaining the figure's size and shape.

📏 Dilation

Resizes a figure while preserving its shape, either increasing or decreasing its size.

🎨 2D Transform Example

This example demonstrates a basic 2D transformation with hover effects.

HTML + CSS - 2D Transform
<div class="box2d">2D Transform</div> <style> .box2d { width: 120px; height: 120px; background: coral; color: white; display: flex; align-items: center; justify-content: center; transition: transform 0.5s ease; } .box2d:hover { transform: rotate(45deg) scale(1.2) translate(20px, 10px); } </style>
✅ What this does:
  • rotate(45deg) - Rotates the element 45 degrees
  • scale(1.2) - Increases size by 20%
  • translate(20px, 10px) - Moves right 20px, down 10px
  • transition - Animates the transformation smoothly

🖱️ Mouse Over Event Example

This example uses JavaScript to change image size when hovering.

HTML + JavaScript - Mouse Events
<!DOCTYPE html> <html> <body> <h1>HTML DOM Events</h1> <h2>The onmouseover Event</h2> <img onmouseover="bigImg(this)" onmouseout="normalImg(this)" border="0" src="smily.jpg" alt="Smiley" width="32" height="32"> <p>The function bigImg() is triggered when the user moves the mouse pointer over the image.</p> <p>The function normalImg() is triggered when the mouse pointer is moved out of the image.</p> <script> function bigImg(x) { x.style.height = "64px"; x.style.width = "64px"; } function normalImg(x) { x.style.height = "32px"; x.style.width = "32px"; } </script> </body> </html>
💡 How it works:
  • onmouseover - Triggers when mouse enters the element
  • onmouseout - Triggers when mouse leaves the element
  • bigImg(this) - Doubles the image size to 64x64px
  • normalImg(this) - Returns to original 32x32px size

🔄 Rotate X-Axis Method

The rotateX() method rotates an element around its X-axis at a given degree.

HTML + CSS - rotateX()
<!DOCTYPE html> <html> <head> <style> div { width: 150px; height: 150px; background-color: yellow; border: 1px solid black; } #myDiv { transform: rotateX(150deg); } </style> </head> <body> <h1>The rotateX() Method</h1> <p>The rotateX() method rotates an element around its X-axis at a given degree.</p> <div>This a normal div element.</div> <div id="myDiv"> This div element is rotated 150 degrees. </div> </body> </html>
✅ Result: The div with id "myDiv" will appear rotated 150 degrees along the X-axis, creating a 3D perspective effect.

🎯 Transform CSS - Multiple Transformations

You can combine multiple transform functions in a single property.

HTML + CSS - Combined Transforms
<style> div.a { width: 150px; height: 80px; background-color: yellow; transform: rotate(20deg); } div.b { width: 150px; height: 80px; background-color: yellow; transform: skewY(20deg); } div.c { width: 150px; height: 80px; background-color: yellow; transform: scaleY(1.5); } </style> <body> <h1>The transform Property</h1> <h2>transform: rotate(20deg):</h2> <div class="a">Hello World!</div> <h2>transform: skewY(20deg):</h2> <div class="b">Hello World!</div> <h2>transform: scaleY(1.5):</h2> <div class="c">Hello World!</div> </body>
💡 Transform Functions:
  • rotate(20deg) - Rotates the element 20 degrees
  • skewY(20deg) - Skews the element along the Y-axis
  • scaleY(1.5) - Scales the element 1.5x vertically

🎯 The Transform-Origin Property

The transform-origin property allows you to change the position of transformed elements.

HTML + CSS - transform-origin
<style> #div1 { position: relative; height: 200px; width: 200px; margin: 100px; padding: 10px; border: 1px solid black; } #div2 { padding: 50px; position: absolute; border: 1px solid black; background-color: red; transform: rotate(45deg); transform-origin: 20% 40%; } </style> <body> <h1>The transform-origin Property</h1> <div id="div1"> <div id="div2">HELLO</div> </div> </body>
✅ What this does:
  • Sets the rotation point to 20% from left, 40% from top
  • Default is 50% 50% (center of element)
  • Can use: percentages, pixels, or keywords (left, right, top, bottom, center)

🎓 Key Takeaways

✅ Translation

Use translate(x, y) to move elements without affecting document flow.

✅ Rotation

Use rotate(), rotateX(), rotateY(), rotateZ() for 2D and 3D rotations.

✅ Scale

Use scale(), scaleX(), scaleY() to resize elements proportionally.

✅ Skew

Use skew(), skewX(), skewY() to slant elements along axes.

✅ Combining Transformations

Multiple transforms can be chained: transform: translate() rotate() scale()

Order matters! Transformations are applied from right to left.

✅ Transform Origin

Use transform-origin to change the pivot point of transformations.

Default is center (50% 50%), but can be set to any position.

💡 Best Practices:
  • Use transition property to animate transformations smoothly
  • Combine transforms in a single property for better performance
  • Remember: transforms don't affect document layout
  • Use 3D transforms (rotateX, rotateY) for depth effects
  • Test browser compatibility for complex 3D transforms
