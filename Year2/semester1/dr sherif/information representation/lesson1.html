Curves in Computer Graphics - Types, Properties, and Implementation

📘 Introduction

In computer graphics, we often need to draw different types of objects onto the screen. Objects are not flat all the time and we need to draw curves many times to draw an object.

💡 What is a Curve?
The curve is an infinitely large set of points. Each point has two neighbors except endpoints.

Canvas Example - Drawing a Half Circle

HTML5 Canvas
<canvas id="arcCanvas" width="300" height="200"></canvas> <script> const ctx = document.getElementById('arcCanvas').getContext('2d'); ctx.beginPath(); ctx.arc(150, 100, 70, 0, Math.PI, false); // Half circle ctx.strokeStyle = 'blue'; ctx.lineWidth = 3; ctx.stroke(); </script>

🎨 Types of Curves

1. Implicit Curves

Set of zeros of a function

2. Explicit Curves

Dependent variable in terms of independent

3. Parametric Curves

Coordinates as functions of a parameter

4. Bezier Curves

Splines with control points

5. B-spline Curves

Flexible curves with local control

🔵 Implicit Curves

An implicit curve or surface is the set of zeros of a function of 2 or 3 variables.

Key Characteristics

  • We use implicit curve functions to define lines and planes
  • Provides no control over tangents at connection points when joining several implicit functions
  • Implicit functions are hard to find for many shapes
  • Use a function that states which points are on and off the curves

Examples

All Lines: Ax + By + C = 0
Any Plane: Ax + By + Cz + D = 0
with constants a, b, c, and d
Sphere: x² + y² + z² - R² = 0
centered at origin with radius R
Circle: X² + Y² - R² = 0
⚠️ Limitations:
  • In three dimensions, f(X,Y,Z) defines a surface
  • Curves in 3D are not so easily represented in implicit form
  • In general, we cannot solve for points that satisfy the implicit form

Canvas Implementation

JavaScript - Drawing with Implicit Function
const canvas = document.getElementById("curveCanvas"); const ctx = canvas.getContext("2d"); function f(x, y) { // Example: circle x² + y² = 100 return x*x + y*y - 100; } ctx.fillStyle = "blue"; for (let px = 0; px < canvas.width; px++) { for (let py = 0; py < canvas.height; py++) { let x = px - 200; let y = py - 200; if (Math.abs(f(x, y)) < 1) { ctx.fillRect(px, py, 1, 1); } } }

🟢 Explicit Curves

Gives the value of one variable (the dependent variable) in terms of the other (the independent variable).

Limitations

  • Do not allow for multiple values for a given argument
  • Cannot describe vertical tangents, as infinite slopes are hard to represent
  • Cannot represent all curves (vertical lines, circles)

Forms

Most familiar form in 2D:
Y = f(x)
where y is dependent, x is independent
Curves:
Z = f(x)
Surfaces:
Z = f(x, y)

🟡 Parametric Curves

Curves have a parametric form called parametric curves. A curve in the plane is said to be parameterized if the set of coordinates on the curve (x, y, z) is represented as a function of a variable t.

The variable t is called a PARAMETER and the relations between x, y, z, and t are called a PARAMETRIC EQUATION

Mathematical Representation

p(t) = f(t), g(t)
or
p(t) = x(t), y(t)

Key Features

  • The parametric form of a curve is a function that assigns a position to values of the free parameters
  • The parametric function is a vector-valued function
  • This example is a 2D curve, so the output is a 2D vector; in 3D it would be a 3-vector
  • It is simple and flexible
  • The parametric form is suitable for representing closed and multivalued curves
  • Each coordinate of a point on a curve is represented as a function of a single parameter
✅ Advantages:
  • Can represent any curve including vertical lines and circles
  • Easy to compute points along the curve
  • Natural for animation (t represents time)
  • Can handle multivalued functions

🟣 Bezier Curves

A Bezier curve is particularly a kind of spline generated from a set of control points by forming a set of polynomial functions.

💡 History: Discovered by the French engineer Pierre Bézier. These functions are computed from the coordinates of the control points.

Mathematical Formula

Σ (k=0 to n) Pi Bi,n(t)

Bi,n(t) = (n choose i) × (1-t)n-i × ti

Characteristics

  • These curves can be generated under the control of other points
  • Tangents by using control points are used to generate curves
  • It is an approximate spline curve
  • A Bezier curve is defined by the defining polygon
  • Has properties that make them highly useful and convenient for curve and surface design

Simple Bezier - Linear (2 Points)

Quadratic Bezier curve is determined by three control points: P0, P1, P2

Cubic Bezier Curve (4 Control Points)

The cubic Bezier curve is determined by four control points: P0, P1, P2, P3

🔷 Properties of Bezier Curves

  1. Bezier curves are widely available and used in various CAD systems, in general graphics packages such as GL
  2. The slope at beginning of the curve is along the line joining the first two control points and the slope at the end of the curve is along the line joining the last two points
  3. Bezier curve always passes through the first and last points
    i.e., p(0) = p0, p(1) = pn
  4. The curves lie entirely within the convex hull formed by the four control points
  5. The degree of polynomial defining the curve segment is one less than the number of defining the polygon
⚠️ Important Note: Bezier curves exhibit global control points, which means moving control points alters the shape of the whole curve.

📐 Different Varieties of Spline Curves

Different varieties of spline curves are used in graphics applications:

1. Hermit Spline
2. Relaxed End Spline
3. Cyclic Spline
4. Anti-Cyclic Spline
5. Normalized Spline

🔶 B-spline Curves

Mathematical Foundation

Σ (i=1 to n+1) ni,k(u) = 1

Where:

  • n+1 = number of control points
  • k = order of B-spline curve

Key Properties

  • The sum of the B-spline basic function at any parametric 'u' equals 1
  • We can add/modify any number of control points to change the shape of the curve without affecting the degree of polynomial
  • Control points affect the shape of the curve only over range of parameter values where its associated basic function is non-zero
  • The polynomial curve has degree (d-1) and Cd-2 continuity over the range of u
  • Each blending function is defined over d sub-intervals of the total range of u
  • The selected set of subintervals endpoints u is referred to as a knot vector
  • The basic function is positive or zero for all parameter values
  • Except for k=1, each basis function has one maximum value
✅ Advantages of B-splines:
  • Local control (moving a control point affects only a portion of the curve)
  • Flexible degree independent of number of control points
  • Always lies within convex hull of control points
  • Numerically stable

⚖️ Curve Types Comparison

Curve Type Advantages Disadvantages
Implicit Easy to define lines/planes Hard to find for complex shapes, no tangent control
Explicit Simple to understand Cannot represent vertical lines, circles
Parametric Flexible, can represent any curve Requires parameter selection
Bezier Intuitive control, smooth curves Global control (moving one point affects entire curve)
B-spline Local control, flexible degree More complex to understand and implement

📝 Summary

✅ Implicit Curves
f(x,y) = 0 or f(x,y,z) = 0
Best for: Lines, planes, circles, spheres
✅ Explicit Curves
y = f(x) or z = f(x,y)
Best for: Simple functions
✅ Parametric Curves
p(t) = [x(t), y(t), z(t)]
Best for: Animation, complex curves
✅ Bezier Curves
Control point based splines
Best for: CAD, graphic design
✅ B-spline Curves
Local control, flexible splines
Best for: Complex modeling
💡 Key Takeaway: The choice of curve representation depends on the application requirements. Parametric curves (especially Bezier and B-splines) are most commonly used in computer graphics due to their flexibility and control.
